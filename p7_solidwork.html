SolidWorks project and Prototype Robotic System
Graphically designed motion toys with specific features using SolidWorks, for each feature finished with three different mechanisms to complete the task;
Completed the toy’s stress, force and motion analysis in hand then reproducing the same analysis by using motion simulation tools in SolidWorks to confirm previous results;



Designed an air-hockey robot in SolidWorks and physically constructed the robot that can move forward, backward, turn all the way round and hit a real air-hockey puck toward destination;
Built hand-controller device by using given circuit kits and wood board to control the motion of air-hockey robot;
